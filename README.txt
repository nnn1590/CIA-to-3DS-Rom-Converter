Create a makerom folder. 
Move makerom and Converter.sh to that folder
Execute Converter.sh with your CIA as an argument and you are good!
Example: ./Converter.sh /path/to/your.cia

makerom: https://github.com/jakcron/Project_CTR/tree/master/makerom
The included makerom binary is built with Ubuntu 18.04.3 LTS x86_64/amd64.
